(async) userLoading(trakt, SB, done)

Source:
  • modules/app/start.js, line 132

Procedure that requests user specific content from trakt.tv. Within this function, it is only checked if new updates are available to request -- if not, all data is already saved in cache. If API requests are needed, requestUpdateDetails is called. A callback is fired when the app can proceed and load the main page.
