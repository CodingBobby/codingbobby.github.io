Task

Modules.Manager. Task

new Task(callback, argsopt, progressopt)

Source:
  • modules/manager/queue.js, line 159

Wraps Promises for easier and more specific use. The Queue will create new Tasks internally but they can be used manually for asynchronous jobs.

Example
new Task(args => sum(args.nums), { nums: [1, 2, 3] })
// argument declaration can also be skipped when not required
new Task(() => sum([1, 2, 3]))
// but it can be usedful in specific cases
new Task(sum, [1, 2, 3])
